# Functional Programming in Haskell

This repository contains examples and exercises related to functional programming using Haskell. The code is organized into different lectures, each focusing on specific concepts and techniques in Haskell.


## Getting Started

To run the Haskell examples, ensure you have GHC (Glasgow Haskell Compiler) installed on your machine. You can run the examples using the following command:

```bash
runhaskell <filename>.hs
```
